📐 提取 Schema 定义

以下 5 类 Schema 定义了从年报中提取的结构化数据格式。每个字段都有明确的类型和描述，Qwen3.7-Plus 会严格按照 Schema 输出 JSON。

🏗️ 六层流水线架构

年报 PDF 从输入到结构化数据输出，经过六层处理。每层职责明确，可独立测试和替换。

🧠 Qwen3.7-Plus 三大核心能力

📖
1M 上下文
200页年报全文可直接送入模型。分块是为了提高精度而非绕过长度限制。
📦
JSON Mode
response_format={"type":"json_object"} 直接返回可解析 JSON，告别正则提取。
📊
多模态理解
年报中的财务表格以 Markdown 格式传入，模型能正确理解行列关系。

🚀 运行方式

              

                
# 1. 安装依赖
pip install pdfplumber openpyxl openai jsonschema

# 2. 配置 API Key
export DASHSCOPE_API_KEY="sk-xxxxxxxxxxxx"

# 3. 提取年报
python main.py --pdf 贵州茅台2024年报.pdf

# 4. 批量提取
python main.py --pdf *.pdf --format excel json

# 5. 指定模型
python main.py --pdf report.pdf --model qwen3.7-plus --verbose
